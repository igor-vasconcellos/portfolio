Services

Front-End Development

Criação de páginas de destino responsivas e otimizadas, com foco em design atrativo e funcionalidade, utilizando HTML, CSS e outras tecnologias de frontend para garantir uma experiência do usuário eficiente e agradável.

UI/UX Design

Desenvolvimento de interfaces de usuário com foco na experiência do usuário (UX), utilizando ferramentas como Figma para criar protótipos, wireframes e designs interativos. Busco sempre equilibrar estética e usabilidade para soluções intuitivas.

Back-End Development

Programação de sistemas robustos e escaláveis utilizando Java, MySQL e outras tecnologias, com foco na criação de funcionalidades eficientes, integração com bancos de dados e processamento de dados de forma segura e otimizada.

TimeControl

O TimeControl é um sistema de controle de horas de trabalho desenvolvido como demanda do meu estágio que busca otimizar o registro de jornada dos funcionários. O sistema permite o registro de entrada, saída e pausas, além de gerar relatórios sobre as horas trabalhadas.

Avaliation System

O Avaliation System foi desenvolvido para uma farmácia, com o objetivo de monitorar a avaliação dos clientes sobre o atendimento e outros serviços. A ferramenta coleta feedbacks dos clientes, permitindo que a farmácia analise a qualidade do atendimento e identifique áreas de melhoria.
